Menu

Live2D

モデリングや表情差分、配信向けの自然な動きづけなどをまとめるセクションです。

ラフのイメージに合わせて、大きなプレビューと短い説明、その下に情報枠を置く構成にしています。

ここには制作範囲や担当工程、使用ソフトなどを入れられます。

制作情報 / ダミー
担当範囲モデリング / 表情差分 / 物理演算 用途配信向け立ち絵 / 動画出演用モデル 補足可動域や見せたい印象に合わせて調整可能

イラスト

キャラクターイラスト、サムネイル、キービジュアルなどを見せるための枠です。

作品説明は長くしすぎず、用途と強みがひと目で分かる量に留めると見やすくなります。

必要ならこの先に作品一覧ページへの導線を入れても自然です。

制作情報 / ダミー
担当範囲キャラクターイラスト / サムネイル / KV 用途配信用、告知用、動画用ビジュアル 補足やわらかい色味や世界観重視のビジュアル設計

デザイン

ロゴ、配信画面、告知画像、サムネイルまわりなどのデザイン制作を紹介する枠です。

ビジュアルの方向性だけでなく、用途や見せたい印象を整理してデザインへ落とし込む流れも伝えられます。

作品単体だけでなく、シリーズものやセット制作の紹介にも向いています。

制作情報 / ダミー
担当範囲ロゴ / 配信素材 / 告知画像 / サムネイル 用途活動ビジュアル整理、配信まわり、SNS展開 補足全体の世界観に合わせた統一感のある設計が可能

作曲

BGM、ジングル、楽曲など、音の制作をまとめるためのセクションです。

今の音楽ポートフォリオへの導線として使うこともでき、ここでは作曲カテゴリ全体の概要を見せる構成にしています。

活動に合った雰囲気づくりや、用途に合わせた音の提案ができることを短く伝えるのに向いています。

制作情報 / ダミー
担当範囲BGM / ジングル / 楽曲制作 用途配信、動画、待機画面、紹介映像など 補足明るい・可愛い・やさしい方向を軸に、幻想的・物語調にも対応

その他できること

・映像制作（紹介映像 / ボカロMV / 歌ってみた など）
・サイト制作
・グラフィック関連の制作サポート

※撮影・3D制作は対応外です
※内容によってはお受けできない場合があります

「こんなことできる？」など、気軽にご相談ください◎

About Naro

作曲、イラスト、デザイン、映像制作などを横断して活動するクリエイターとして、 やわらかい世界観と見せ方の設計を大切にしながら制作しています。

このポートフォリオでは、分野ごとの作品だけでなく、全体の雰囲気や対応できる制作範囲もまとめて見られる構成を意識しています。
